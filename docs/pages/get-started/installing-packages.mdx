# Installing Packages

When adding packages, you will need to point to the correct workspace

### YARN

to add packages to the Frontend Environment dependencies

<Code>yarn workspace @create-permaweb-app/frontend add (package-name)</Code>

to add packages to the backend Environment dependencies

<Code>yarn workspace @create-permaweb-app/backend add (package-name)</Code>
